# Support

## Getting Help

- **Documentation**: [docs/USAGE.md](docs/USAGE.md) - Complete CLI and feature reference
- **Issues**: [GitHub Issues](https://github.com/hyperlight-dev/hyperagent/issues) - Bug reports and feature requests

## Reporting Bugs

Please include:
- Hyperagent version (`hyperagent --version`)
- Operating system and hypervisor (KVM/MSHV/WHP)
- Steps to reproduce
- Expected vs actual behavior
- Relevant logs or error messages

## Feature Requests

Open an issue describing:
- Your use case
- Proposed solution
- Alternatives you've considered

## Security Issues

See [SECURITY.md](SECURITY.md) for reporting security vulnerabilities.

**Do not report security issues through public GitHub issues.**

## Microsoft Support Policy

Support for **HyperAgent** is limited to the resources listed above.
